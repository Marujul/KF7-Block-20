📖 Kosakata Kegiatan Sehari-hari

日本語 Romaji Arti

📚 Pola Kalimat

Pola: Subjek + Partikel + Kata Kerja (-masu)

わたしは がっこうへ いきます。
Watashi wa gakkō e ikimasu.
Arti: Saya pergi ke sekolah.

Pola: Waktu + Partikel + Kegiatan

あさ 7じに おきます。
Asa shichi-ji ni okimasu.
Arti: Saya bangun tidur jam 7 pagi.

Pola: Kata Benda + を + Kata Kerja

あさごはんを たべます。
Asagohan o tabemasu.
Arti: Saya makan sarapan.

Partikel Penting

Partikel Penggunaan Contoh
は (wa) Penanda topik kalimat わたしは がくせい です。
を (o) Penanda objek langsung ごはんを たべます。
に (ni) Penanda waktu/tempat tujuan 7じに おきます。
へ (e) Penanda arah/tujuan がっこうへ いきます。
で (de) Penanda tempat kegiatan きょうしつで べんきょうします。
と (to) Penanda "dengan" ともだちと あそびます。

💬 Contoh Dialog

Dialog 1: Pagi Hari

A: おはようございます。おきましたか。
Ohayō gozaimasu. Okimashita ka?
Arti: Selamat pagi. Apakah kamu sudah bangun?
B: はい、おきました。かおを あらって、はを みがきました。
Hai, okimashita. Kao o aratte, ha o migakimashita.
Arti: Ya, sudah bangun. Saya sudah mencuci muka dan menggosok gigi.
A: じゃ、あさごはんを たべましょう。
Ja, asagohan o tabemashō.
Arti: Kalau begitu, mari kita makan sarapan.
